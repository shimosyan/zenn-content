## 🙆‍♂️ Terraform で使用するコマンド

以下のコマンドをよく使います。

### Terraform を初期化し、必要な Provider をインストールする

```bash
terraform init
```

### Terraform のコードを整形する

```bash
terraform fmt
```

### Terraform の設定ファイルが正しいかチェックし、Okta への反映内容を計画する

```bash
terraform plan
```

### チェックの通った設定ファイルを本番の Okta に適用する

```bash
terraform apply
```
